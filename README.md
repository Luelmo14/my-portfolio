# My Portfolio

This is a personal portfolio website built with React, Vite, and shadcn/ui.

## Technologies Used

- React
- TypeScript
- Vite
- shadcn/ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
